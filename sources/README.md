# Source material (raw)

Drop ebook / text files here for the cut-up library: **PDF, EPUB, TXT, DOCX** — mixed formats fine.

These raw files are **processed offline** (extract → clean → segment → tag) into the app's
optimised corpus. They are **gitignored** — never committed to the public repo (size + copyright).

Optional: group files in subfolders by poet/collection to help tagging, e.g.
```
sources/
  bukowski/
    love-is-a-dog-from-hell.epub
  plath/
    ariel.pdf
  my-own/
    notebook-2025.txt
```
